\section*{概要}
私は「高度な技術実装」と「社会課題の解決」の両立を信条とするエンジニアです。高専在学中、自社プロジェクト「MEKOU」を立ち上げ、分散型AIインフラの開発や介護ロボティクスへの応用に取り組んできました。

\section*{主な実績とスキルセット}
\begin{itemize}
    \item EJPT
    \item DCON 2026 本選出場
    \item 様々なプロジェクト開発、共同開発ゲーム開発なども経験
    \item 介護職員初任者研修の資格取得中
    \item CTFやハッカソンなどのコンテスト参加経験も豊富
    \item Rust, C系、 JAVA, TSなどのプログラミング言語、AI関連技術、SLAM、ロボティクスなど幅広い技術スタックを保有
\end{itemize}
\subsection*{1. サイバーセキュリティと国際性}
\begin{itemize}
    \item \textbf{eJPT (Junior Penetration Tester) 取得}: 英語によるペネトレーションテスト実技試験に合格。脆弱性診断からエクスプロイトまでの一連のプロセスを2週間で習得。
    \item \textbf{グローバル経験}: アメリカでのホームステイを通じ、多文化環境下でのコミュニケーション能力を育成。
\end{itemize}

\subsection*{2. 開発実績と受賞歴}
\begin{itemize}
    \item \textbf{DCON 2026 本選出場}: ディープラーニングを活用した高専生起業コンテストにて、ビジネスモデルと技術実装の両面で評価され本選進出。
    \item \textbf{MEKOU Engineの開発}: Rust言語およびECSを用いた独自の分散型AI計算基盤の設計・実装。
    \item \textbf{SLAM・ロボティクス}: 介護用二足歩行ロボットの実現に向けたSLAM計算およびリアルタイム映像伝送システムの構築。
\end{itemize}

\subsection*{3. 社会実装へのアプローチ}
\begin{itemize}
    \item \textbf{介護職員初任者研修}: 技術のみならず、実際の介護現場のニーズを深く理解するため、専門資格の取得を通じたドメイン知識の習得。
\end{itemize}

\section*{志望動機と今後の展望}
長岡技術科学大学の「実務訓練」および「フロンティアコース」を通じ、現在取り組んでいるMEKOUプロジェクトを学術的・産業的に昇華させたいと考えています。特に、高度な情報経営システムと電気電子情報の知見を融合させ、実社会で真に機能する自律型インフラの構築を目指します。
